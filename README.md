# 🏥 Medical Insurance Cost Prediction using Multiple Linear Regression

This project predicts **medical insurance charges** based on a person's demographic and health-related information using the **Multiple Linear Regression** algorithm.<br><br>

## 📌 Project Overview

This machine learning project estimates medical insurance charges using features such as **Age, Sex, BMI, Children, Smoker Status, and Region**. The dataset is preprocessed using **Label Encoding** and **One-Hot Encoding** before training the model.<br><br>

## 🚀 Features

✔ Data Preprocessing<br>
✔ Label Encoding<br>
✔ One-Hot Encoding<br>
✔ Train-Test Split<br>
✔ Multiple Linear Regression Model<br>
✔ Insurance Charge Prediction<br>
✔ Model Evaluation using R² Score<br><br>

## 📊 Dataset Features

- Age<br>
- Sex<br>
- BMI (Body Mass Index)<br>
- Children<br>
- Smoker Status<br>
- Region<br>
- Charges (Target Variable)<br><br>

## ⚙️ Workflow

1. Import Required Libraries<br>
2. Load Dataset<br>
3. Data Preprocessing<br>
4. Label Encoding for Binary Features<br>
5. One-Hot Encoding for Region<br>
6. Train-Test Split<br>
7. Train Multiple Linear Regression Model<br>
8. Predict Insurance Charges<br>
9. Evaluate Model Performance using R² Score<br><br>
